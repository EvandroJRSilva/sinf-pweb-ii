1. Dos grupos
    1. Os alunos deverão se reunir em grupos de até 5 pessoas.
      1. Caso todos os grupos já tenham se formado e sobrar até 2 alunos, estes poderão ingressar em algum dos grupos já formado, de preferência um em cada grupo.
      2. Se estiverem sobrando 3 alunos ou mais, estes devem se juntar para formar um novo grupo.
  2. Do trabalho
    1. O trabalho consistirá em uma aplicação web a ser desenvolvida durante o semestre. Os grupos deverão reunir-se para decidirem o tema do projeto.
    2. O trabalho terá 3 fases:
      1. Fase 1: Desenvolvimento de uma interface gráfica. Neste ponto os grupos estão liberados para utilizar frameworks e outras ferramentas.
      2. Fase 2: Implementação dos seguintes conceitos:
        • Comunicação síncrona e assíncrona (Fetch API).
        • Web Storage API.
        • Cookie Store API ou HTTP cookies.
        • HTTP Authentication ou Web Authentication API.
        • HTTPS e certificados digitais.
        • Validação de formulários.
      3. Fase 3: Integração com banco de dados
        • Ajustar o sistema atual para o uso de frameworks.
        • Modelar os dados, criar e conectar com o banco de daods.
        • Consultas SQL.
    3. A avaliação de cada fase ocorrerá da seguinte forma:
      1. Fase 1: Análise da interface gráfica desenvolvida. Serão favorecidas as interfaces mais complexas e completas em relação ao sistema escolhido pelo grupo.
      2. Fase 2: Análise da apresentação da aplicação desenvolvida até então. Pontuarão melhor os grupos em que todos os membros apresentem bem (com desenvoltura e seguros) os aspectos do sistema, principalmente sobre os detalhes técnicos.
      3. Fase 3: Análise da apresentação final. Pontuarão melhor os grupos em que todos os membros apresentem bem todos os aspectos técnicos do sistema.
    4. As datas previstas para cada fase são:
      1. Fase 1: 10/10/2025, o código fonte ou link para o repositório.
      2. Fase 2: 05/11/25 e 06/11/25. Metade dos grupos apresentando em cada data. O tempo de apresentação será calculado posteriormente, quando todos os grupos tiverem sido definidos.
      3. Fase 3: 03/12/25 e 04/12/25. Metade dos grupos apresentando em cada data. O tempo de apresentação será calculado posteriormente, quando todos os grupos tiverem sido definidos.
