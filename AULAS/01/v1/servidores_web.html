Servidores Web

Apache HTTP Server
Servidor web de código aberto mais utilizado no mundo, desenvolvido pela Apache Software Foundation.
Arquitetura: Baseada em processos/threads (um para cada conexão)
Módulos: Sistema extensível com diversos módulos disponíveis
Compatibilidade: Suporte a múltiplas plataformas e linguagens
Nginx
Servidor web de alto desempenho, também usado como proxy reverso, balanceador de carga e cache HTTP.
Arquitetura: Orientada a eventos, assíncrona (um processo atende múltiplas conexões)
Performance: Excelente para conteúdo estático e muitas conexões simultâneas
Uso: Ideal para sites de alto tráfego e aplicações modernas
