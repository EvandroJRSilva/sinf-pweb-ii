Protocolo HTTP e HTTPS

HTTP (Hypertext Transfer Protocol)
Protocolo de comunicação utilizado para transferir dados na web, baseado no modelo cliente-servidor.
Sem estado (stateless): Cada requisição é independente
Métodos: GET, POST, PUT, DELETE, etc.
Segurança: Dados trafegam em texto plano (não criptografados)
HTTPS (HTTP Secure)
Versão segura do HTTP que utiliza criptografia SSL/TLS para proteger a comunicação.
Criptografia: Dados trafegam criptografados
Certificados: Utiliza certificados SSL/TLS para autenticação
Confiança: Garante integridade e autenticidade dos dados
Diferença entre HTTP e HTTPS: comunicação não criptografada vs. criptografada
