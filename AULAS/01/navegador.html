Funcionamento Básico do Navegador

Como o navegador processa uma página
1
Requisição
O navegador envia uma requisição HTTP para o servidor solicitando um recurso (HTML, CSS, JS, imagens).
2
Parsing
O navegador analisa o HTML recebido e constrói a árvore DOM (Document Object Model).
3
Renderização
Combina o DOM com o CSSOM (CSS Object Model) para criar a árvore de renderização.
4
Layout
Calcula a posição e tamanho exatos de cada elemento na página.
5
Pintura
Converte cada nó da árvore de renderização em pixels na tela.
