Exemplo Prático: Requisição Síncrona em JavaScript

⚠️ Aviso: Este exemplo demonstra uma requisição síncrona, que é considerada uma má prática em aplicações web modernas pois bloqueia a interface do usuário. Use apenas para fins educacionais!

Como funciona:

Quando você clicar no botão abaixo, o navegador fará uma requisição síncrona para uma API pública. Durante esse tempo, a interface ficará completamente bloqueada e você não conseguirá interagir com a página.

Clique no botão acima para ver o resultado...

Código do Exemplo:

// Função que demonstra requisição síncrona (BLOQUEANTE)
function exemploSincrono() {
    const resultado = document.getElementById('resultado');
    const btn = document.getElementById('btnSincrono');
    
    // Desabilita o botão e mostra status
    btn.disabled = true;
    resultado.innerHTML = '<p class="loading">Fazendo requisição síncrona... A interface está BLOQUEADA!</p>';
    
    try {
        // Cria XMLHttpRequest em modo síncrono (terceiro parâmetro = false)
        const xhr = new XMLHttpRequest();
        xhr.open('GET', 'https://jsonplaceholder.typicode.com/posts/1', false);
        
        // Esta linha BLOQUEIA a execução até receber resposta
        xhr.send();
        
        if (xhr.status === 200) {
            const dados = JSON.parse(xhr.responseText);
            resultado.innerHTML = `
                <h3>✅ Requisição concluída!</h3>
                <p><strong>Título:</strong> ${dados.title}</p>
                <p><strong>Conteúdo:</strong> ${dados.body}</p>
                <p><em>Note que a interface ficou bloqueada durante toda a requisição.</em></p>
            `;
        } else {
            resultado.innerHTML = '<p style="color: red;">❌ Erro na requisição: ' + xhr.status + '</p>';
        }
    } catch (error) {
        resultado.innerHTML = '<p style="color: red;">❌ Erro: ' + error.message + '</p>';
    } finally {
        btn.disabled = false;
    }
}

// Função para demonstrar que a interface fica bloqueada
function tentarClicar() {
    alert('Se você conseguiu ver este alerta durante a requisição, significa que a interface NÃO estava bloqueada!');
}

Observações Importantes:

  • Bloqueio da Interface: Durante a requisição síncrona, você não conseguirá clicar em nenhum botão ou interagir com a página.
  • Experiência do Usuário: O navegador pode parecer "travado" ou não responsivo.
  • Uso Moderno: Requisições síncronas são desencorajadas em aplicações web modernas.
  • Alternativa: Use requisições assíncronas com fetch(), Promises ou async/await.
