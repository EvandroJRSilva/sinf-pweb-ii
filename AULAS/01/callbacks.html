Introdução aos Callbacks

O que são Callbacks?
Callbacks são funções passadas como argumentos para outras funções, que serão executadas após a conclusão de uma operação assíncrona ou em um momento específico.
// Exemplo básico de callback em JavaScript
function buscarDados(url, callback) {
  const xhr = new XMLHttpRequest();
  xhr.open('GET', url, true);

  xhr.onload = function() {
    if (xhr.status === 200) {
      const dados = JSON.parse(xhr.responseText);
      callback(null, dados);
    }
  };

  xhr.send();
}

buscarDados('https://api.exemplo.com/dados', function(erro, dados) {
  if (!erro) console.log(dados);
});
Não-bloqueante: Permite que o código continue executando durante operações assíncronas
Flexibilidade: Personaliza o comportamento após a conclusão de uma operação
